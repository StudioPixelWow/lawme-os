## Rule #1 — one hero object

The Today Focus scene now has exactly one hero: the full-width navy
mission-control object. One objective (דיון הוכחות — כהן נ׳ לוי), ONE
dominant number ("48 דק׳" in display type, with a quiet "בעוד" above),
one readiness ring, one דינו insight, one champagne action. The
prepared-work column that used to compete beside it became a quiet
manifest row of chips beneath the object. The gating facts shrank to one
caption line. Nothing else in the first viewport speaks at that volume.

## Rule #2 — two accents

- **Attention strip:** every figure is now ink; only the critical item
  (2 מועדים בסיכון) keeps the gold wash + red figure. The secondary
  "עבור למקטע" micro-lines were removed — the items are one clean line.
- **Team bench:** gauges are machined ink; only genuine overload speaks
  red (דניאל, מיכל). Presence dots and state labels went neutral except
  urgent. All copper/blue/green accents removed from the section.
- **Client communication:** channel icons went from three colored washes
  to neutral sunken containers; urgency remains the only color.
- **Utility rail:** the communication block's icons neutralized.

## Rules #3–#4 — merge, collapse, prioritize by frequency

- **Leads** collapsed to the one hot lead (₪80,000, gold edge) + a
  "עוד 2 בצנרת" disclosure. Occasional-use content no longer holds
  permanent space.
- **Client communication** items now open on selection only — the page
  no longer auto-expands a reply.
- **Featured matter** densified: tighter padding and rhythm, facts
  seated on a hairline — less air, same information, more operational.
