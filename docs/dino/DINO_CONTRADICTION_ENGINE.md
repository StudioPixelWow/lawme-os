# Contradiction Engine

> Epic 3A — Dino Legal Intelligence Orchestration Engine. POC over the
> synthetic employment-law corpus. Deterministic core; provider-independent;
> safe auditable artifacts only (no model chain-of-thought).

contradictions/contradiction-engine.ts. Surfaces opposite conclusions,
limiting exceptions, contrary case law, and (where metadata allows)
conflicting versions / later treatment. Each record: proposition A + source
A, proposition B + source B, type, direct/apparent, distinction factors,
authority comparison, temporal comparison, human-review recommendation,
resolution status, materiality. Conflicts are never hidden; unresolved
material conflicts reduce confidence and route to humans. POC limitation:
temporal/version conflicts need real metadata (declared).
