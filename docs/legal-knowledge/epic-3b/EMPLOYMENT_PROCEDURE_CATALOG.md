# Employment Procedure Catalog (Epic 3B Triad, Pillar C)

11 connected procedures (`procedure/catalog.ts`):
pregnancy_dismissal (fully detailed, 9 stages intake→appeal),
severance_claim, hearing_before_dismissal, pre_dismissal_dispute
(prior notice), wage_overtime_claim, pension_rights_claim,
discrimination_claim, harassment_complaint, regional_labor_court_civil,
appeal_to_national_labor_court, settlement_enforcement.

The pregnancy-dismissal procedure implements the founder's example graph:
intake → confirm employment relationship/duration → confirm pregnancy
timing → confirm employer knowledge → confirm permit status → preserve
dismissal documents/communications → assess hearing → identify urgent
remedies → determine forum → demand/application → deadlines → file →
interim relief → evidence → judgment → appeal/enforcement. Each stage links
to חוק עבודת נשים §9 (real permalink laws/2001135), required evidence,
deadlines (with calculation notes + "to verify" honesty), and
human-approval-gated actions.
